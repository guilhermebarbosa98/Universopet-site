# Universopet-site
# 🐾 Universo Pet — Sistema Web

O **Universo Pet** é um sistema web desenvolvido com o objetivo de simular um site completo de um petshop, oferecendo uma navegação intuitiva e moderna para apresentação de produtos e serviços voltados ao cuidado de animais de estimação.

## 📌 Sobre o projeto

Este projeto foi desenvolvido como parte da disciplina de **Sistemas Web da PUCRS (Pontifícia Universidade Católica do Rio Grande do Sul)**, com foco na aplicação prática de conceitos fundamentais de desenvolvimento front-end.

A aplicação permite que usuários naveguem entre diferentes páginas, visualizem produtos, conheçam serviços oferecidos e realizem o agendamento de atendimentos por meio de um formulário interativo.

## 🚀 Funcionalidades

* 📄 Navegação entre múltiplas páginas (Home, Produtos, Serviços e Contatos)
* 🛍️ Exibição de produtos organizados por categorias
* 🧸 Seção de brinquedos com layout moderno em cards
* 🛠️ Seção de serviços com opção de agendamento
* 📝 Formulário completo para cadastro e agendamento de serviços
* 🎠 Carrossel de imagens na página inicial
* 🔔 Interações dinâmicas com JavaScript (mensagens, alertas e eventos)
* 📱 Layout responsivo utilizando Bootstrap

## 🧰 Tecnologias utilizadas

* HTML5
* CSS3
* Bootstrap 5
* JavaScript

## 🎨 Interface

A interface foi desenvolvida com foco em usabilidade e experiência do usuário, utilizando componentes do Bootstrap como navbar responsiva, cards e carrossel de imagens. Além disso, foram aplicados estilos personalizados em CSS para aprimorar a identidade visual do sistema.

## ⚙️ Estrutura do projeto

O projeto é organizado em múltiplas páginas HTML interligadas, com separação de responsabilidades entre estilos (CSS) e comportamentos (JavaScript), garantindo maior organização e escalabilidade.

## 💡 Objetivo

O principal objetivo deste projeto é consolidar conhecimentos em desenvolvimento web, aplicando conceitos de estruturação de páginas, estilização moderna e interatividade com JavaScript.

## 📈 Possíveis melhorias futuras
